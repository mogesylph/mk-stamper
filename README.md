# mk-stamper

mk-stamper は、YouTube の歌枠や長時間動画を見ながら、曲ごとのタイムスタンプを記録・編集するための個人用ツールです。

柚羽まくらさんという Vsinger の歌枠のタイムスタンプを記録しやすくするために作りました。

また、Codex を使ったバイブコーディングの検証として制作したものでもあります。

このリポジトリは個人制作の非公式ツールです。柚羽まくらさん本人、所属先、YouTube とは関係ありません。

## Features

- YouTube URL または videoId から動画を読み込み
- 現在の再生位置を開始時刻・終了時刻として登録
- 曲名・歌手名の記録と編集
- 曲マスタにない曲の確認
- タイムスタンプリストの TSV コピー
- ブラウザの localStorage による作業状態の保存

## Concept

このアプリでは、YouTube プレイヤーを最も重要な UI として扱います。

画面遷移や複雑な設定を増やさず、動画を見ながらすばやく記録できることを優先しています。詳しい方針は [docs/concept.md](docs/concept.md) にまとめています。

## Development

アプリ本体は `app` 以下にあります。

```bash
cd app
npm install
npm run dev
```

本番ビルドは以下で確認できます。

```bash
cd app
npm run build
```

## Data

登録したタイムスタンプや動画の復元情報は、ブラウザの localStorage に保存されます。

曲マスタは `app/public/songlist.tsv` を参照します。
